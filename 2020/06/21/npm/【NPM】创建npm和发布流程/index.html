点击 Generate token，生成token，然后复制生成的串码，粘贴到 Password 后面，粘贴之后，cmd上会显示空白的，没事，其实已经粘贴上了，目前npm密码是这样设定的，然后enter，切到 email，填入邮箱；然后就登录ok了。
